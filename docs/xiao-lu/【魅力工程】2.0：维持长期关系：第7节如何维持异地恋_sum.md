-   异地恋需要沟通和信任
    1.  吵架要升级为沟通，精神陪伴很重要
    2.  适合异地恋的人要独立、善于沟通、专一
    3.  同城相处是解决异地恋问题的关键
-   异地恋需要信任和空间
    1.  信任是感情的基础，猜忌会伤害感情
    2.  光明正大的相处，不要束缚对方

-   填满空闲时间，丰富生活，分享点滴，加强感情

    2.  通过通讯工具表达情感，增强感情温度

    2.  吵架要有意义，提高沟通，增进了解
    3.  渗透朋友圈，提高信任度，制定见面计划
-   制定见面计划，建立预期，制造浪漫

    2.  取消干扰，保证见面质量
