## 下载

### Docker

```
docker pull apachecn0/love-course-2020
docker run -tid -p <port>:80 apachecn0/love-course-2020
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install love-course-2020
love-course-2020 <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g love-course-2020
love-course-2020 <port>
# 访问 http://localhost:{port} 查看文档
```
